# Golf Distance Simulator（ドライバー&ウッド / ミドルアイアン）

## 概要
このアプリは、ゴルフショットの**キャリー**と**トータル**を比較表示するシミュレーターです。  
2画面（A/B）で入力条件を並べて計算し、弾道グラフと距離結果を同時に確認できます。

- ページ構成
  - `index.html` : ドライバー&ウッド
  - `middle-iron.html` : ミドルアイアン
  - `pro.html` : Pro版案内（Coming Soon）

---

## 主な機能
- A/B の条件比較（ヘッドスピード、ミート率、打ち出し角、スピン、風）
- 2D弾道表示（飛行軌道 + 着地後挙動）
- キャリー / トータル / 最大到達点の表示
- A→B / B→A 入力コピー
- ページモード別の入力レンジ検証

---

## 入力レンジ
### ドライバー&ウッド（`index.html`）
- ヘッドスピード: `25.0〜60.0 m/s`
- ミート率: `1.30〜1.56`
- 打ち出し角: `8.0〜25.0 °`
- スピン: `1500〜5000 rpm`
- 風向風速: `-7.0〜+7.0 m/s`

### ミドルアイアン（`middle-iron.html`）
- ヘッドスピード: `15.0〜40.0 m/s`
- ミート率: `1.10〜1.30`
- 打ち出し角: `10.0〜30.0 °`
- スピン: `1500〜10000 rpm`
- 風向風速: `-7.0〜+7.0 m/s`

---

## 計算モデル（実装ベース）
本アプリは鉛直面内の2次元モデルで、速度ベクトルに対して重力・抗力・揚力を与えて時間積分します。

- 飛行パート
  - 速度依存の抗力係数 `Cd(Re)`
  - スピンファクターから求める揚力係数 `Cl(S)`
  - スピン減衰を考慮
  - 地面到達時は線形補間で着地点を算出

- 着地後パート
  - バウンドと接地減衰
  - 着弾時のスピン影響を地面反応に反映
  - 転がりは `vx` の二乗依存モデル（符号付き）

> 注意: 本ツールは実測を置き換えるものではなく、比較検討向けの推定モデルです。

## 開発/確認方法
ローカルで簡易サーバを立ててブラウザ確認できます。

```bash
python -m http.server 4173
```

- ドライバー&ウッド: `http://localhost:4173/index.html`
- ミドルアイアン: `http://localhost:4173/middle-iron.html`
- Pro: `http://localhost:4173/pro.html`
